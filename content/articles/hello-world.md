# Welcome to NexusCMS

This is your first article! NexusCMS is a modern, headless CMS powered by Eleventy, combining the simplicity of static site generation with a user-friendly admin interface.

## What is NexusCMS?

NexusCMS provides:
- **Fast Static Sites** - Generated with Eleventy for lightning-fast performance
- **Easy Content Management** - Web-based admin dashboard for creating and editing articles
- **GitHub Integration** - Content stored in your repository
- **Image Storage** - Supabase integration for media management
- **Automatic Deployment** - Deploy hooks for continuous deployment

## Getting Started

1. Navigate to the `/admin/` directory to access the content management interface
2. Configure your GitHub personal access token in Settings
3. Start creating articles using the Content Editor
4. Publish directly to your repository

## Features

- **Live Preview** - See your content as you write it
- **SEO Optimization** - Built-in meta tags and descriptions
- **Reading Time** - Automatically calculated for each article
- **RSS Feed** - Automatic RSS feed generation
- **Responsive Design** - Works great on all devices

## Next Steps

- Customize the theme in `index.html`
- Configure your site settings in `_data/site.json`
- Add more markdown files to `content/articles/`
- Deploy to your hosting platform

Happy blogging! 🚀
